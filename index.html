✕Закрыть
Как открыть и проверить тему:
1) Локально: сохраните как miniapp.html и откройте в браузере.
2) Тема на компьютере: Chrome → F12 → ⋮ → More tools → Rendering → «Emulate CSS media feature prefers-color-scheme» → light или dark.
3) GitHub Pages: загрузите репозиторий на GitHub, в настройках репо включите Pages (Source: main, /root). Ссылка будет https://ВАШ_ЛОГИН.github.io/ИМЯ_РЕПО/ (для index.html можно не писать в URL).
4) В Telegram: создайте бота (@BotFather), в кнопке меню укажите type: web_app, url: ваша ссылка с GitHub Pages. Откройте мини-приложение из бота — тема подстроится под тёмную/светлую тему в настройках Telegram (Настройки → Тема оформления).
5) Проверка темы по ссылке: откройте ?theme=light или ?theme=dark в конце URL, чтобы принудительно выбрать тему (например, на GitHub Pages: .../index.html?theme=light).
Здесь всё собрано в структуру, которую вы надиктовали. Ссылки пока плейсхолдеры — подключим реальные.
